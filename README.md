O AlphaPay for WooCommerce é a melhor forma de receber pagamentos online por cartão de crédito, 
sendo possível o cliente fazer todo o pagamento sem sair da sua loja WooCommerce.

## Criação e configuração do usuário
Antes de você utilizar o AlphaPay é importante você verificar se o seu usuário e chave de API está habilitado para a integração
segue URL de configuração do usuário AlphaPay:
https://www.alphapay.com.br/

## Compatibilidade

PHP >= 5.4+<br/>
Wordpress >= 4.x+

## Instalação

Envie os arquivos do plugin para a pasta wp-content/plugins, ou instale usando o instalador de plugins do WordPress.<br/>
Ative o plugin.<br/>
Entre com: Endereço da Live API e KEY da Live API fornecidos pela plataforma AlphaPay.
